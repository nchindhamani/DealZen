Key Technologies
================
Backend:        FastAPI, Python 3.9+, Weaviate Client, OpenAI SDK
Frontend:       React 18, Vite, Tailwind CSS, Axios
Database:       Weaviate (Vector Database with Hybrid Search)
AI Model:       GPT-4o (for RAG answer generation)
Deployment:     Docker (Weaviate), uvicorn (Backend), Vite (Frontend)


Quick Commands
==============
Setup Everything:       ./setup_all.sh
Start Weaviate:         ./start_weaviate.sh
Setup Backend Only:     ./setup_backend.sh
Setup Frontend Only:    ./setup_frontend.sh

Ingest Data:            cd scripts && python ingest_data.py
Run Backend:            cd backend && uvicorn app.main:app --reload
Run Frontend:           cd frontend && npm run dev

Access Application:     http://localhost:5173
API Documentation:      http://localhost:8000/docs
Weaviate Console:       http://localhost:8080
